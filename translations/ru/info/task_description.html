Вам часто приходится работать с различными геометрическими фигурами и узнавать их параметры - периметр, площадь, объем. Устав делать это вручную, вы решили автоматизировать процесс. Для этого вам нужно написать программу, с помощью которой можно будет создавать экземпляры класса Figure и применять к ним различные методы:
Figure().perimeter() (возвращает периметр фигуры)
Figure().area() (возвращает площадь фигуры)
Figure().volume() (возвращает объем фигуры)
Список фигур, с которыми вы работаете:
"circle" --> Figure("circle", R), где R - радиус ( 0.1 <= R <= 1000.0)
"triangle" --> Figure("triangle", a, b, c), где a, b, c - длины сторон треугольника ( 0.1 <= a, b, c <= 1000.0)
"rectangle" --> Figure("rectangle", a, b), где a, b - длины сторон прямоугольника ( 0.1 <= a, b <= 1000.0)
"pentagon" --> Figure("pentagon", a), где a - длина стороны равностороннего пятиугольника ( 0.1 <= a <= 1000.0)
"hexagon" --> Figure("hexagon", a), где a - длина стороны равностороннего шестиугольника ( 0.1 <= a <= 1000.0)
"cube" --> Figure("circle", a), где a - длина ребра куба ( 0.1 <= a <= 1000.0)

Так как все фигуры, кроме куба, не имеют объема, вам нужно будет вернуть 0, если метод volume() будет применен к любой фигуре, кроме куба. Если результат получается без дробной части - верните его как int(), в ином случае - округлите до 2 знаков после запятой.
В этой миссии вам может помочь такой шаблон проектирования, как Strategy.

Примеры: 

Figure("circle", 10).area() == 314.16
Figure("triangle", 3, 4, 5).volume() == 0
Figure("rectangle", 1.5, 2.5).area() == 3.75
Figure("pentagon", 500).perimeter() == 2500
Figure("hexagon", 100.003).perimeter() == 600.02
Figure("cube", 20).volume() == 8000


Входные данные: операторы и выражения, использующие класс Figure.

Выходные данные: периметр, площадь или объем (число).

Как это используется: Для анализа геометрических объектов.

Предусловие: Все данные корректны.
